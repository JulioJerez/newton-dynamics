Overview

dBrain is a neural network library integrated into Newton Dynamics, designed for both supervised learning and reinforcement learning applications in physics simulation. It provides GPU acceleration via OpenCL with automatic CPU fallback.

Quick Example

#include <ndBrain.h>

// Create a simple feed-forward network
ndBrain brain;
brain.AddLayer(new ndBrainLayerLinear(inputSize, 128));
brain.AddLayer(new ndBrainLayerActivationRelu(128));
brain.AddLayer(new ndBrainLayerLinear(128, 64));
brain.AddLayer(new ndBrainLayerActivationRelu(64));
brain.AddLayer(new ndBrainLayerLinear(64, outputSize));
brain.InitWeights();

// Forward inference
ndBrainVector input(inputSize);
ndBrainVector output(outputSize);
// ... fill input ...
brain.MakePrediction(input, output);

Use Cases

dBrain is designed for machine learning applications in physics simulation:

  • Robot Locomotion - Train articulated characters to walk, run, and balance
  • Vehicle Control - Learn driving policies for cars, boats, aircraft
  • Industrial Robots - Inverse kinematics via neural networks
  • Game AI - NPCs that learn from physics interactions
  • Control Systems - PID replacement with learned controllers

Header Files

Main include for dBrain:

#include <ndBrain.h>        // Core network and layers
#include <ndBrainTrainer.h> // Training infrastructure
#include <ndBrainAgent.h>   // Reinforcement learning agents

The dBrain library is located in sdk/dBrain/ with GPU implementations in sdk/dBrain/dOpenCl/ and CPU fallback in sdk/dBrain/cpuEmulation/.
